# Welcome to LiberChat repo!
年齢関係なく使える、オープンソースのチャットサイト、LiberChatのソースコードです。
## 貢献規約
最低限守るべきこと
- セミコロンは禁止。
- コンポーネントとクラスはパスカルケースにする。それ以外の変数はキャメルケースにする。
- スネークケースは使用禁止。
- プルリクのタイトルはデフォルトを使わない。自分の変更をわかるようにしっかり書く。
